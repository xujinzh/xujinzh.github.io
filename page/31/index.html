Following manual login, Teamviewer is able to connect at the login screen until the next reboot.） 这是因为，teamviewer 服务需要用户登陆系统后才能打开，因此，可以通过修改配置让 teamviewer 服务自动打开，这样就可以再重启 Ubuntu 系统后，直接用 teamviewer 连接。具体的 1sudo vim /etc/gdm3/custom.conf 把 1WaylandEnable=false 注销掉 然后，重启系统 1sudo reboot 重启 Ubuntu 系统后，就可以使用 teamviewer 进行连接。
