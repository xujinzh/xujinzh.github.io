ip йҰ–йғЁж ЎйӘҢе’ҢгҖҒtcp/udp йҰ–йғЁж ЎйӘҢе’ҢиғҪеӨҹеҲӨж–ӯзҪ‘з»ңдј иҫ“дёӯжҳҜеҗҰеҮәзҺ°дёўеҢ…гҖҒй”ҷеҢ…зӯүгҖӮдёӢйқўеҲҶеҲ«д»Ӣз»Қе®ғд»¬зҡ„и®Ўз®—ж–№жі•е’Ң python е®һзҺ°гҖӮ

ip йҰ–йғЁж ЎйӘҢе’Ң

ip йҰ–йғЁж ЎйӘҢе’ҢжҳҜй’ҲеҜ№ ip ж•°жҚ®еҢ…зҡ„еҢ…еӨҙ20еӯ—иҠӮйғЁеҲҶиҝӣиЎҢж ЎйӘҢгҖӮеҒҮи®ҫжҲ‘д»¬жңүеҰӮдёӢ20еӯ—иҠӮзҡ„ ip еҢ…еӨҙж•°жҚ®пјҲ16иҝӣеҲ¶ж јејҸпјүпјҡ

  • 45пјҡ4иЎЁзӨә ip зүҲжң¬еҸ·пјҢиҝҷйҮҢжҳҜ ipv4пјҢ5иЎЁзӨәйҰ–йғЁж•°жҚ®й•ҝеәҰпјҢдёә5дёӘ32жҜ”зү№пјҢеҚі20еӯ—иҠӮпјӣ
  • 00 30: иЎЁзӨә ip еҢ…жҖ»й•ҝеәҰдёә 0x0030=48еӯ—иҠӮпјҢеҮҸеҺ»еӨҙйғЁ20еӯ—иҠӮпјҢж•°жҚ®йғЁеҲҶдёә28еӯ—иҠӮпјӣ
  • 80 4cпјҡиЎЁзӨәж ҮиҜҶз¬Ұпјӣ
  • 40 00пјҡ3жҜ”зү№ж Үеҝ—е’Ң13жҜ”зү№зүҮеҒҸз§»йҮҸпјӣ
  • 80 06пјҡ80иЎЁзӨәttlеҖј0x80=128пјҢ06иЎЁзӨә tcp еҚҸи®®пјҲ0x11=17иЎЁзӨәudpеҚҸи®®пјүпјӣ
  • b5 2eпјҡиЎЁзӨә ip йҰ–йғЁж ЎйӘҢе’Ңпјӣ
  • d3 43 11 7bпјҡиЎЁзӨәжәҗең°еқҖ0xd3, 0x43, 0x11, 0x7b = (211, 67, 17, 123)пјӣ
  • cb 51 15 3dпјҡиЎЁзӨәзӣ®зҡ„ең°еқҖ0xcb, 0x51, 0x15, 0x3d = (203, 81, 21, 61)пјӣ

ip йҰ–йғЁж ЎйӘҢе’Ңз®—жі•

и®Ўз®— ip йҰ–йғЁж ЎйӘҢе’ҢпјҢ

  1. е°Ҷ ip еҢ…еӨҙ20еӯ—иҠӮзҡ„ ip йҰ–йғЁж ЎйӘҢе’Ң2еӯ—иҠӮжӣҝжҚўдёә 00 00пјҢжӣҝжҚўеҗҺеҰӮдёӢжүҖзӨәпјҡ

  2. еҜ№20еӯ—иҠӮиҝӣиЎҢдёӨеӯ—иҠӮд»ҘеӨ§з«Ҝж–№ејҸзҙҜеҠ жұӮе’Ңпјҡ

  3. еҰӮжһңе’ҢеӨ§дәҺ2еӯ—иҠӮж—¶пјҢеҲҷе’ҢеҖјдёӯй«ҳ2еӯ—иҠӮеҠ дёҠдҪҺ2еӯ—иҠӮзҡ„еҖјдҪңдёәжңҖз»Ҳзҡ„е’ҢеҖјпјҡ

  4. еҜ№жңҖз»Ҳзҡ„е’ҢеҖјжұӮеҸҚпјҢеҚіе…ідәҺ2еӯ—иҠӮзҡ„иЎҘпјҡ

жӯӨж—¶пјҢеҫ—еҲ°зҡ„еҖј 0xb52e е°ұжҳҜдёҠйқўзҡ„ ip йҰ–йғЁж ЎйӘҢе’Ң b5 2e

ip йҰ–йғЁж ЎйӘҢе’Ңзҡ„ python е®һзҺ°

tcp/udp йҰ–йғЁж ЎйӘҢе’Ң

tcp е’Ң udp зҡ„йҰ–йғЁж ЎйӘҢе’Ңз®—жі•жҳҜдёҖиҮҙзҡ„пјҢеҸӘжҳҜж•°жҚ®дёӯзҡ„еҚҸи®®ж ҮиҜҶдёҚеҗҢпјҢtcpдёә0x06пјҢudpдёә0x11гҖӮеҗҢж—¶пјҢйҷӨдәҶйңҖиҰҒеҜ№ж•°жҚ®ж·»еҠ дјӘйҰ–йғЁе’Ңж ЎйӘҢе’ҢдҪҚзҪ®дёҚеҗҢеӨ–пјҢе®ғд»¬зҡ„и®Ўз®—з®—жі•е’Ң ip йҰ–йғЁж ЎйӘҢе’ҢжҳҜдёҖиҮҙзҡ„гҖӮдёӢйқўд»Ҙ udp дёәдҫӢгҖӮ

ж•°жҚ®ж јејҸеҰӮдёӢпјҡ

е…¶дёӯпјҡ

  • udp дјӘйҰ–йғЁпјҲ12еӯ—иҠӮпјүпјҡ4еӯ—иҠӮжәҗең°еқҖ + 4еӯ—иҠӮзӣ®зҡ„ең°еқҖ + 0x00 + 0x11пјҲudp еҚҸи®®гҖӮ tcp еҚҸи®®еҸ–еҖј0x06пјү+ 2еӯ—иҠӮ udp й•ҝеәҰ;
  • udp йҰ–йғЁпјҲ8еӯ—иҠӮпјүпјҡ2еӯ—иҠӮжәҗз«ҜеҸЈ + 2еӯ—иҠӮзӣ®зҡ„з«ҜеҸЈ + 2еӯ—иҠӮ udp й•ҝеәҰ + 2еӯ—иҠӮж ЎйӘҢе’Ңпјӣ

жіЁж„Ҹпјҡ

  • udp ж•°жҚ®жҠҘзҡ„жҖ»й•ҝеәҰеҢ…жӢ¬ udp йҰ–йғЁ + иҪҪиҚ·ж•°жҚ®пјӣ
  • udp дјӘйҰ–йғЁеҠ е…Ҙж ЎйӘҢзҡ„зӣ®зҡ„жҳҜдёәдәҶеҶҚж¬Ўж ёеҜ№ж•°жҚ®жҠҘжҳҜеҗҰеҲ°иҫҫжӯЈзЎ®зҡ„зӣ®зҡ„ең°пјҢе№¶йҳІжӯў ip ж¬әйӘ—ж”»еҮ» (spoofing);

ж•°жҚ®дҫӢеӯҗпјҡ

  • еүҚйқў20еӯ—иҠӮдёә ip еҢ…еӨҙж•°жҚ®пјӣ
  • d3 43 11 7bпјҡдёә ip жәҗең°еқҖпјӣ
  • cb 51 15 3dпјҡдёә ip зӣ®зҡ„ең°еқҖпјӣ
  • f8 3dпјҡдёәжәҗз«ҜеҸЈпјӣ
  • 30 30: дёәзӣ®зҡ„з«ҜеҸЈпјӣ
  • 00 12пјҡдёә udp еҢ…й•ҝеәҰпјҢ0x0012 = 18пјҢиЎЁзӨә udp йҰ–йғЁ + иҪҪиҚ·ж•°жҚ®пјҢд»Һжәҗз«ҜеҸЈеҲ°ж•°жҚ®еҢ…жң«е°ҫпјӣ
  • b1 2dпјҡдёә udp ж ЎйӘҢе’Ң;
  • 55 aa ејҖе§ӢеҲ°жң«е°ҫйғҪжҳҜиҪҪиҚ·ж•°жҚ®гҖӮ

udp йҰ–йғЁж ЎйӘҢе’Ңз®—жі•

udp йҰ–йғЁж ЎйӘҢе’Ңзҡ„з®—жі•жӯҘйӘӨеҰӮдёӢпјҲжӯҘйӘӨ3гҖҒ4гҖҒ5>еҗҢip йҰ–йғЁж ЎйӘҢе’Ңз®—жі•пјүпјҡ

  5. иҝӣиЎҢдёӨеӯ—иҠӮд»ҘеӨ§з«Ҝж–№ејҸзҙҜеҠ жұӮе’ҢпјҢеҰӮжһңе’ҢеӨ§дәҺ2еӯ—иҠӮж—¶пјҢеҲҷе’ҢеҖјдёӯй«ҳ2еӯ—иҠӮеҠ дёҠдҪҺ2еӯ—иҠӮзҡ„еҖјдҪңдёәжңҖз»Ҳзҡ„е’ҢеҖј:

udp йҰ–йғЁж ЎйӘҢе’Ң python е®һзҺ°

з®Җдҫҝж–№жЎҲ

е…¶е®һеңЁдёҠйқўи®Ўз®— ip йҰ–йғЁж ЎйӘҢе’Ңж—¶пјҢйҰ–е…Ҳе°Ҷ ip йҰ–йғЁж ЎйӘҢе’ҢзҪ®дҪҚ 00 00 зҡ„йӮЈдёҖжӯҘеҸҜд»ҘзңҒз•ҘпјҢеҸӘйңҖиҰҒеңЁжңҖз»Ҳи®Ўз®—зҡ„з»“жһңиҝӣиЎҢдҪҺ2гҖҒ1еӯ—иҠӮ(sum & 0xFFFF)е’ҢдҪҺ4гҖҒ3еӯ—иҠӮ(sum >> 16)з»„жҲҗзҡ„ж•ҙж•°зӣёеҠ еҚіеҸҜпјҢеҰӮжһңеҖјдёә 0xFFFFпјҢеҲҷиҜҙжҳҺ ip йҰ–йғЁж ЎйӘҢе’ҢжӯЈзЎ®гҖӮ

еҜ№дәҺ udp/tcp ж ЎйӘҢдёҖж ·пјҢеӣ дёәз®—жі•дёҺ ip йҰ–йғЁж ЎйӘҢзӣёеҗҢгҖӮжӯЈеёёжһ„йҖ дјӘ udpпјҢзңҒеҺ»йҰ–йғЁж ЎйӘҢе’ҢзҪ®дёә 00 00 зҡ„дёҖжӯҘпјҢжңҖеҗҺи®Ўз®—дҪҺ2гҖҒ1еӯ—иҠӮ(sum & 0xFFFF)е’ҢдҪҺ4гҖҒ3еӯ—иҠӮ(sum >> 16)з»„жҲҗзҡ„ж•ҙж•°зӣёеҠ еҚіеҸҜпјҢеҰӮжһңеҖјдёә 0xFFFFпјҢеҲҷиҜҙжҳҺ udp/tcp йҰ–йғЁж ЎйӘҢе’ҢжӯЈзЎ®гҖӮ
