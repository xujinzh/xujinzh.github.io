我这里有个需求，就是服务器会因为断电等原因关机、重启，导致 IP 地址更改，无法通过 SSH 连接，也不能及时知道电脑何时开机。这里通过撰写 Python 代码，自动检测 IP、开机自动邮件通知。本篇以 Ubuntu 为例。
