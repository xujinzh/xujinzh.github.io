Python 是一种广泛使用的解释型、高级和通用的编程语言。除了 numpy 提供科学计算外，还有 sympy 用于数学符号计算。

sympy 的安装非常简单，使用 pip install sympy 即可完成。更详细的安装教程请参考 sympy Installation 官网。
