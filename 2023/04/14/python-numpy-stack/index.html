Python 中 numpy 数组拼接

发表于2023-04-14|更新于2024-01-30|technologypython
|字数总计:786|阅读时长:3分钟|阅读量:

numpy 是 Python 科学计算库，效率非常高，其自带的数据结构 ndarray 能够非常方便的处理多维数据，在 numpy 中定义了很多多维数据拼接的方法，本篇简要介绍它们。

hstack, vstack, dstack

hstack 水平拼接

hstack 能够沿水平方向拼接两个行数相同的多维数组。如下

array([[1, 2],
       [3, 4],
       [5, 6]])

array([[11, 12],
       [13, 14],
       [15, 16]])

array([[ 1,  2, 11, 12],
       [ 3,  4, 13, 14],
       [ 5,  6, 15, 16]])

((3, 2), (3, 2), (3, 4))

vstack 垂直拼接

vstack 能够沿垂直方向拼接两个列数相同的多维数组。如下

array([[1, 2, 3],
       [4, 5, 6]])

array([[11, 12, 13],
       [14, 15, 16]])

array([[ 1,  2,  3],
       [ 4,  5,  6],
       [11, 12, 13],
       [14, 15, 16]])

((2, 3), (2, 3), (4, 3))

dstack 深度拼接

dstack 能够沿着深度方向拼接两个行数和列数都相同的多维数组。如下

array([[1, 2, 3],
       [4, 5, 6]])

array([[11, 12, 13],
       [14, 15, 16]])

array([[[ 1, 11],
        [ 2, 12],
        [ 3, 13]],

       [[ 4, 14],
        [ 5, 15],
        [ 6, 16]]])

((2, 3), (2, 3), (2, 3, 2))

concatenate 指定拼接方向

concatenate 能够指定拼接的方式，但只能指定水平和垂直方向，不包含深度方向。如下

array([[1, 2, 3],
       [4, 5, 6]])

array([[11, 12, 13],
       [14, 15, 16]])

array([[ 1,  2,  3],
       [ 4,  5,  6],
       [11, 12, 13],
       [14, 15, 16]])

((2, 3), (2, 3), (4, 3))

array([[ 1,  2,  3, 11, 12, 13],
       [ 4,  5,  6, 14, 15, 16]])

((2, 3), (2, 3), (2, 6))

stack 维度扩充

stack 维度扩展，把多个二维的矩阵扩展到三维，如下

(1024, 1024)


(1024, 1024)


(1024, 1024, 3)


(1024, 3, 1024)

(1024, 1024, 3)


(3, 1024, 1024)

(1024, 1024, 3)
