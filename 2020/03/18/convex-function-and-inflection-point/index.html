тЄИтЄйТЋ░тњїТІљуѓ╣

тЈЉУАеС║ј2020-03-18|ТЏ┤Тќ░С║ј2023-09-10|mathsmathematical analysis
|тГЌТЋ░Тђ╗У«А:4.6k|жўЁУ»╗ТЌХжЋ┐:NaN:aNтѕєжњЪ|жўЁУ»╗жЄЈ:

тЄИтЄйТЋ░№╝ѕConvex function№╝ЅтњїТІљуѓ╣№╝ѕInflection point№╝Ѕсђѓ ТГцжЃетѕєСИ║тЈѓУђЃ тЇјСИютИѕУїЃтцДтГдсђіТЋ░тГдтѕєТъљСИітєїсђІсђѓС╗ЦСИІтєЁт«╣СИ║тГдС╣аСИГуџёжЄЇуѓ╣Тђ╗у╗Њсђѓ

тЄИтЄйТЋ░СИђУѕгућеТЮЦУ»ЂТўјСИЇуГЅт╝Ј№╝ЏТ│еТёЈтидтЈ│т»╝ТЋ░тњїтЄИтЄйТЋ░т«џС╣ЅтЁгт╝ЈуџёУЂћу│╗№╝ЏтЄИтЄйТЋ░уџёТђДУ┤еТ»ћУЙЃтЦй№╝їтюеТю║тЎетГдС╣аСИГТў»уљєТЃ│уџёС╗БС╗итЄйТЋ░№╝їтЏаСИ║тЦйТ▒ѓТюђт░Јтђ╝уѓ╣№╝ЏТ│еТёЈТІљуѓ╣СИГуџёжъЇуѓ╣№╝ѕТў»СИђжўХт»╝ТЋ░уГЅС║ј0уџёуѓ╣№╝їтЇ│ТІљуѓ╣СИГуџёжЕ╗уѓ╣Тѕќуе│т«џуѓ╣№╝Ѕсђѓ

тЄИтЄйТЋ░

т«џС╣Ѕ1 У«Й$f$СИ║т«џС╣Ѕтюетї║жЌ┤$I$СИіуџётЄйТЋ░№╝їУІЦт»╣$I$СИіуџёС╗╗ТёЈСИцуѓ╣$x_1, x_2$тњїС╗╗ТёЈт«ъТЋ░$\lambda\in(0,1)$Тђ╗ТюЅ
$$
f(\lambda x_1 + (1-\lambda) x_2)\leq \lambda f(x_1) + (1 - \lambda)f(x_2),
$$
тѕЎуД░$f$СИ║$I$СИіуџётЄИтЄйТЋ░сђѓтЈЇС╣І№╝їтдѓТъюТђ╗ТюЅ
$$
f(\lambda x_1 + (1 - \lambda) x_2)\geq \lambda f(x_1) + (1 - \lambda) f(x_2),
$$
тѕЎуД░$f$СИ║$I$СИіуџётЄ╣тЄйТЋ░сђѓ

тЈдтцќ№╝ї

  1. тдѓТъюСИіжЮбуџёСИЇуГЅт╝ЈСИ║СИЦТа╝СИЇуГЅт╝Ј№╝їжѓБС╣ѕуД░СИЦТа╝тЄИтЄйТЋ░тњїСИЦТа╝тЄ╣тЄйТЋ░сђѓ
  2. тдѓТъю$-f$СИ║тї║жЌ┤$I$СИіуџётЄ╣тЄйТЋ░№╝їжѓБС╣ѕ$f$СИ║тї║жЌ┤$I$СИіуџётЄИтЄйТЋ░сђѓ

т╝Ћуљє $f$СИ║$I$СИіуџётЄИтЄйТЋ░уџётЁЁУдЂТЮАС╗ХТў»т»╣С║ј$I$СИіуџёС╗╗ТёЈСИЅуѓ╣$x_1 < x_2 < x_3$№╝їТђ╗ТюЅ
$$
\frac{f(x_2) - f(x_1)}{x_2 - x_1} \leq \frac{f(x_3) - f(x_2)}{x_3 - x_2}.
$$
тљїТаитю░№╝ї$f$СИ║$I$СИіуџётЄИтЄйТЋ░уџётЁЁУдЂТЮАС╗ХТў»т»╣С║ј$I$СИіуџёС╗╗ТёЈСИЅуѓ╣$x_1 < x_2 < x_3$№╝їТюЅ
$$
\frac{f(x_2) - f(x_1)}{x_2 - x_1} \leq \frac{f(x_3) - f(x_1)}{x_3 - x_1} \leq \frac{f(x_3) - f(x_2)}{x_3 - x_2}.
$$
Т│еТёЈ№╝їУ┐ЎжЄїТіі$x_2, x_3$уюІуЮђСИђСИфтЈўжЄЈТЌХ№╝їтЄйТЋ░$\frac{f(x_3) - f(x_1)}{x_3 - x_1}$Тў»СИђСИфтбътЄйТЋ░№╝їтЈ»С╗ЦтѕЕућетЇЋУ░ЃТюЅуЋїт«џуљє№╝їтЈќТъЂжЎљ

т«џуљє6.14 У«Й$f$СИ║тї║жЌ┤$I$СИіуџётЈ»т»╝тЄйТЋ░№╝їтѕЎСИІУ┐░У«║ТќГС║њуЏИуГЅС╗и№╝џ

  1. $f$СИ║$I$СИіуџётбътЄйТЋ░№╝Џ

  2. $f^{\prime}$СИ║$I$СИіуџётбътЄйТЋ░№╝Џ

  3. т»╣$I$СИіуџёС╗╗ТёЈСИцуѓ╣$x_1, x_2$ТюЅ
    $$
    f(x_2) \geq f(x_1) + f^{\prime}(x_1)(x_2 - x_1).
    $$
    У«║ТќГ3уџётЄаСйЋТёЈС╣Ѕт░▒Тў»У»┤ТЏ▓у║┐$y=f(x)$Тђ╗Тў»тюет«ЃуџёС╗╗СИђтѕЄу║┐СИіТќ╣сђѓ

т«џуљє 6.15 У«Й$f$СИ║тї║жЌ┤$I$СИіуџёС║їжўХтЈ»т»╝тЄйТЋ░№╝їтѕЎтюе$I$СИі$f$СИ║тЄИтЄйТЋ░№╝ѕтЄ╣тЄйТЋ░№╝ЅуџётЁЁУдЂТЮАС╗ХТў»
$$
f^{(2)}(x) \geq 0 \ (f^{(2)}(x) \leq 0), \ x \in I.
$$
т«џуљє УІЦтЄйТЋ░$f$СИ║т«џС╣Ѕтюет╝ђтї║жЌ┤$(a, b)$СИіуџётЈ»т»╝уџётЄИтЄйТЋ░№╝ѕтЄ╣тЄйТЋ░№╝Ѕ№╝їтѕЎ$x_0 \in (a, b)$СИ║$f$уџёТъЂт░Јтђ╝уѓ╣№╝ѕТъЂтцДтђ╝уѓ╣№╝ЅуџётЁЁУдЂТЮАС╗ХТў»$x_0$СИ║$f$уџёуе│т«џуѓ╣№╝їтЇ│$f^{\prime}(x_0) = 0$.

т«џуљє У«ЙтЄйТЋ░$f(x)$СИ║т╝ђтї║жЌ┤$(a, b)$СИіуџётЄИтЄйТЋ░№╝їСИЇТЂњСИ║тИИТЋ░№╝їтѕЎ$f(x)$СИЇтЈќТюђтцДтђ╝сђѓ

РђІ тЄИтЄйТЋ░уџёТюђтцДтђ╝тЈфУЃйтюеуФ»уѓ╣тцётЈќтЙЌсђѓтЇ│№╝їУІЦтЄйТЋ░$f(x)$Тў»жЌГтї║жЌ┤$[a, b]$СИіуџётЄИуџёУ┐ъу╗ГтЄйТЋ░№╝їжѓБС╣ѕ
$$
f(x) \leq \max{f(a), f(b)}.
$$
УЕ╣ТБ«№╝ѕJensen№╝ЅСИЇуГЅт╝Ј УІЦ$f$СИ║$[a, b]$СИіуџётЄИтЄйТЋ░№╝їтѕЎт»╣С╗╗ТёЈ$x_i \in [a, b], \lambda_i > 0 \ (i = 1, 2, \cdots, n), \ \sum^n_{i = 1} \lambda_i = 1$№╝їТюЅ
$$
f(\sum^n_{i=1} \lambda_i x_i) \leq \sum^n_{i=1} \lambda_i f(x_i).
$$
т╝Ћуљє У«Й$f$СИ║т╝ђтї║жЌ┤$I$тєЁуџётЄИтЄйТЋ░№╝ѕтЄ╣тЄйТЋ░№╝Ѕ№╝їтѕЎ$f$тюе$I$тєЁС╗╗СИђуѓ╣$x_0$жЃйтГўтюетидсђЂтЈ│т»╝ТЋ░сђѓС╗јУђї$f$тюе$x_0$У┐ъу╗Гсђѓ

ТІљуѓ╣

т«џС╣Ѕ2 У«ЙТЏ▓у║┐$y = f(x)$тюеуѓ╣$(x_0, f(x_0))$тцёТюЅуЕ┐У┐ЄТЏ▓у║┐уџётѕЄу║┐№╝їСИћтюетѕЄуѓ╣У┐ЉТЌЂ№╝їТЏ▓у║┐тюетѕЄу║┐уџёСИцСЙДтѕєтѕФТў»СИЦТа╝тЄИтњїСИЦТа╝тЄ╣уџё№╝їУ┐ЎТЌХуД░уѓ╣$(x_0, f(x_0))$СИ║ТЏ▓у║┐$y = f(x)$уџёТІљуѓ╣сђѓ

ТІљуѓ╣Тў»тЄ╣тњїтЄИТЏ▓у║┐уџётѕєуЋїуѓ╣сђѓ

т«џуљє6.16 УІЦ$f$тюе$x_0$С║їжўХтЈ»т»╝№╝їтѕЎ$(x_0, f(x_0))$СИ║ТЏ▓у║┐$y = f(x)$уџёТІљуѓ╣уџёт┐ЁУдЂТЮАС╗ХТў»$f^{(2)} = 0$.

т«џуљє6.17 У«Й$f$тюе$x_0$тЈ»т»╝№╝їтюеТЪљжбєтЪЪ$U^o(x_0)$СИіС║їжўХтЈ»т»╝сђѓУІЦтюе$U^o_{+}(x_0)$тњї$U^o_{-}(x_0)$СИі$f^{(2)}$уџёугдтЈиуЏИтЈЇ№╝їтѕЎ$(x_0, f(x_0))$СИ║ТЏ▓у║┐$y = f(x)$уџёТІљуѓ╣сђѓ

Т│еТёЈ№╝џУІЦ$(x_0, f(x_0))$Тў»ТЏ▓у║┐$y = f(x)$уџёСИђСИфТІљуѓ╣№╝ї$y = f(x)$тюе$x_0$уџёт»╝ТЋ░СИЇСИђт«џтГўтюе№╝їтдѓ$y = \sqrt[3]{x}$тюе$x = 0$сђѓ

т«џуљє тдѓСИІСИЅСИфжЎѕУ┐░ТѕљуФІсђѓ

  1. УІЦ$f$СИ║тЄИтЄйТЋ░№╝ї$\lambda$СИ║жЮъУ┤Ът«ъТЋ░№╝їтѕЎ$\lambda f$СИ║тЄИтЄйТЋ░№╝Џ
  2. УІЦ$f, g$тЮЄСИ║тЄИтЄйТЋ░№╝їтѕЎ$f + g$СИ║тЄИтЄйТЋ░№╝Џ
  3. УІЦ$f$СИ║тї║жЌ┤$I$СИітЄИтЄйТЋ░№╝ї$g$СИ║$J \supset f(I)$СИітЄИтбътЄйТЋ░№╝їтѕЎ$g \circ f$СИ║$I$СИітЄИтЄйТЋ░сђѓ

т«џуљє У«Й$f$СИ║тї║жЌ┤$I$СИіСИЦТа╝тЄИтЄйТЋ░№╝ї$x_0 \in I$СИ║$f$уџёТъЂт░Јтђ╝уѓ╣№╝їтѕЎ$x_0$СИ║$f$тюе$I$СИітћ»СИђуџёТъЂт░Јтђ╝уѓ╣сђѓ

т«џуљє УІЦ$f, g$тЮЄСИ║тї║жЌ┤$I$СИітЄИтЄйТЋ░№╝їтѕЎ$F(x) = \max {f(x), g(x)}$С╣ЪТў»$I$СИітЄИтЄйТЋ░сђѓ

тЄаСИфжЄЇУдЂСИЇуГЅт╝Ј

тЮЄтђ╝СИЇуГЅт╝Ј У«Й$a_i > 0 \ (i = 1, 2, \cdots, n)$№╝їТюЅ
$$
\frac{n}{\frac{1}{a_1} + \frac{1}{a_2} + \cdots + \frac{1}{a_n}} \leq \sqrt[n]{a_1 a_2 \cdots a_n} \leq \frac{a_1 + a_2 + \cdots + a_n}{n}
$$
У»ЂТўј№╝їТъёжђатЄйТЋ░$f(x) = \ln(x)$тЈ»У»ЂТўјуггС║їСИфСИЇуГЅтЈи№╝їуёХтљјућетЈўтїќ$x = \frac{1}{t}$тњїуггС║їСИфСИЇуГЅтЈиУ»ЂТўјуггСИђСИфСИЇуГЅтЈисђѓ

Young СИЇуГЅт╝Ј У«Й$a, b, p, q$тЮЄТў»ТГБт«ъТЋ░№╝їСИћ$\frac{1}{p} + \frac{1}{q} = 1$№╝їтѕЎ
$$
ab \leq \frac{a^p}{p} + \frac{b^q}{q}
$$
уГЅтЈиТѕљуФІтйЊСИћС╗ЁтйЊ $a^p = b^q$.

У»ЂТўј№╝їтѕЕућетЄйТЋ░$y = e^x$уџётЄИТђД
$$
ab = e^{\ln(a)} e^{\ln(b)} = e^{\frac{1}{p} \ln(a^p) + \frac{1}{q} \ln(b^q)} \leq \frac{1}{p} e^{\ln(a^p)} + \frac{1}{q} e^{\ln(b^q)} = \frac{a^p}{p} + \frac{b^q}{q}.
$$
тЈ»С╗ЦућеУ»ЦСИЇуГЅт╝ЈУ»ЂТўјтдѓСИІуџёHolderСИЇуГЅт╝Јсђѓ

Holder СИЇуГЅт╝Ј У«Й$a_i, b_i > 0 \ (i = 1, 2, \cdots, n)$№╝їТюЅ
$$
\sum^n_{i = 1} a_i b_i \leq (\sum^n_{i = 1} a^p_i)^{\frac{1}{p}}(\sum^n_{i = 1} b^q_i)^{\frac{1}{q}}
$$
тЁХСИГ$p > 1, q > 1, \frac{1}{p} + \frac{1}{q} = 1$.

У»ЂТўј№╝їтЈќ$a = \frac{a_k}{(\sum^n_{i = 1}a^p_i)^{\frac{1}{p}}}$, $b = \frac{b_k}{(\sum^n_{i = 1}b^q_i)^{\frac{1}{q}}}$ №╝їт╣ХтИдтЁЦYoungСИЇуГЅт╝Јсђѓ
