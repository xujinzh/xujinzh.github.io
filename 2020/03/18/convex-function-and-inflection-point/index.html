еЗЄеЗљжХ∞

еЃЪдєЙ1 иЃЊ$f$дЄЇеЃЪдєЙеЬ®еМЇйЧі$I$дЄКзЪДеЗљжХ∞пЉМиЛ•еѓє$I$дЄКзЪДдїїжДПдЄ§зВє$x_1, x_2$еТМдїїжДПеЃЮжХ∞$\lambda\in(0,1)$жАїжЬЙ

еИЩзІ∞$f$дЄЇ$I$дЄКзЪДеЗЄеЗљжХ∞гАВеПНдєЛпЉМе¶ВжЮЬжАїжЬЙ
$$
f(\lambda x_1 + (1 - \lambda) x_2)\geq \lambda f(x_1) + (1 - \lambda) f(x_2),
$$
еИЩзІ∞$f$дЄЇ$I$дЄКзЪДеЗєеЗљжХ∞гАВ

еП¶е§ЦпЉМ

  1. е¶ВжЮЬдЄКйЭҐзЪДдЄНз≠ЙеЉПдЄЇдЄ•ж†ЉдЄНз≠ЙеЉПпЉМйВ£дєИзІ∞дЄ•ж†ЉеЗЄеЗљжХ∞еТМдЄ•ж†ЉеЗєеЗљжХ∞гАВ
  2. е¶ВжЮЬ$-f$дЄЇеМЇйЧі$I$дЄКзЪДеЗєеЗљжХ∞пЉМйВ£дєИ$f$дЄЇеМЇйЧі$I$дЄКзЪДеЗЄеЗљжХ∞гАВ

еЉХзРЖ $f$дЄЇ$I$дЄКзЪДеЗЄеЗљжХ∞зЪДеЕЕи¶БжЭ°дїґжШѓеѓєдЇО$I$дЄКзЪДдїїжДПдЄЙзВє$x_1 < x_2 < x_3$пЉМжАїжЬЙ
$$
\frac{f(x_2) - f(x_1)}{x_2 - x_1} \leq \frac{f(x_3) - f(x_2)}{x_3 - x_2}.
$$
еРМж†ЈеЬ∞пЉМ$f$дЄЇ$I$дЄКзЪДеЗЄеЗљжХ∞зЪДеЕЕи¶БжЭ°дїґжШѓеѓєдЇО$I$дЄКзЪДдїїжДПдЄЙзВє$x_1 < x_2 < x_3$пЉМжЬЙ
$$
\frac{f(x_2) - f(x_1)}{x_2 - x_1} \leq \frac{f(x_3) - f(x_1)}{x_3 - x_1} \leq \frac{f(x_3) - f(x_2)}{x_3 - x_2}.
$$
ж≥®жДПпЉМињЩйЗМжКК$x_2, x_3$зЬЛзЭАдЄАдЄ™еПШйЗПжЧґпЉМеЗљжХ∞$\frac{f(x_3) - f(x_1)}{x_3 - x_1}$жШѓдЄАдЄ™еҐЮеЗљжХ∞пЉМеПѓдї•еИ©зФ®еНХи∞ГжЬЙзХМеЃЪзРЖпЉМеПЦжЮБйЩР

еЃЪзРЖ6.14 иЃЊ$f$дЄЇеМЇйЧі$I$дЄКзЪДеПѓеѓЉеЗљжХ∞пЉМеИЩдЄЛињ∞иЃЇжЦ≠дЇТзЫЄз≠ЙдїЈпЉЪ

  1. $f$дЄЇ$I$дЄКзЪДеҐЮеЗљжХ∞пЉЫ

  2. $f^{\prime}$дЄЇ$I$дЄКзЪДеҐЮеЗљжХ∞пЉЫ

  3. еѓє$I$дЄКзЪДдїїжДПдЄ§зВє$x_1, x_2$жЬЙ
    $$
    f(x_2) \geq f(x_1) + f^{\prime}(x_1)(x_2 - x_1).
    $$
    иЃЇжЦ≠3зЪДеЗ†дљХжДПдєЙе∞±жШѓиѓіжЫ≤зЇњ$y=f(x)$жАїжШѓеЬ®еЃГзЪДдїїдЄАеИЗзЇњдЄКжЦєгАВ

еЃЪзРЖ 6.15 иЃЊ$f$дЄЇеМЇйЧі$I$дЄКзЪДдЇМйШґеПѓеѓЉеЗљжХ∞пЉМеИЩеЬ®$I$дЄК$f$дЄЇеЗЄеЗљжХ∞пЉИеЗєеЗљжХ∞пЉЙзЪДеЕЕи¶БжЭ°дїґжШѓ
$$
f^{(2)}(x) \geq 0 \ (f^{(2)}(x) \leq 0), \ x \in I.
$$
еЃЪзРЖ иЛ•еЗљжХ∞$f$дЄЇеЃЪдєЙеЬ®еЉАеМЇйЧі$(a, b)$дЄКзЪДеПѓеѓЉзЪДеЗЄеЗљжХ∞пЉИеЗєеЗљжХ∞пЉЙпЉМеИЩ$x_0 \in (a, b)$дЄЇ$f$зЪДжЮБе∞ПеАЉзВєпЉИжЮБе§ІеАЉзВєпЉЙзЪДеЕЕи¶БжЭ°дїґжШѓ$x_0$дЄЇ$f$зЪДз®≥еЃЪзВєпЉМеН≥$f^{\prime}(x_0) = 0$.

еЃЪзРЖ иЃЊеЗљжХ∞$f(x)$дЄЇеЉАеМЇйЧі$(a, b)$дЄКзЪДеЗЄеЗљжХ∞пЉМдЄНжБТдЄЇеЄЄжХ∞пЉМеИЩ$f(x)$дЄНеПЦжЬАе§ІеАЉгАВ

вАЛ еЗЄеЗљжХ∞зЪДжЬАе§ІеАЉеП™иГљеЬ®зЂѓзВєе§ДеПЦеЊЧгАВеН≥пЉМиЛ•еЗљжХ∞$f(x)$жШѓйЧ≠еМЇйЧі$[a, b]$дЄКзЪДеЗЄзЪДињЮзї≠еЗљжХ∞пЉМйВ£дєИ
$$
f(x) \leq \max{f(a), f(b)}.
$$
и©єж£ЃпЉИJensenпЉЙдЄНз≠ЙеЉП иЛ•$f$дЄЇ$[a, b]$дЄКзЪДеЗЄеЗљжХ∞пЉМеИЩеѓєдїїжДП$x_i \in [a, b], \lambda_i > 0 \ (i = 1, 2, \cdots, n), \ \sum^n_{i = 1} \lambda_i = 1$пЉМжЬЙ
$$
f(\sum^n_{i=1} \lambda_i x_i) \leq \sum^n_{i=1} \lambda_i f(x_i).
$$
еЉХзРЖ иЃЊ$f$дЄЇеЉАеМЇйЧі$I$еЖЕзЪДеЗЄеЗљжХ∞пЉИеЗєеЗљжХ∞пЉЙпЉМеИЩ$f$еЬ®$I$еЖЕдїїдЄАзВє$x_0$йГље≠ШеЬ®еЈ¶гАБеП≥еѓЉжХ∞гАВдїОиАМ$f$еЬ®$x_0$ињЮзї≠гАВ

еЃЪдєЙ2 иЃЊжЫ≤зЇњ$y = f(x)$еЬ®зВє$(x_0, f(x_0))$е§ДжЬЙз©њињЗжЫ≤зЇњзЪДеИЗзЇњпЉМдЄФеЬ®еИЗзВєињСжЧБпЉМжЫ≤зЇњеЬ®еИЗзЇњзЪДдЄ§дЊІеИЖеИЂжШѓдЄ•ж†ЉеЗЄеТМдЄ•ж†ЉеЗєзЪДпЉМињЩжЧґзІ∞зВє$(x_0, f(x_0))$дЄЇжЫ≤зЇњ$y = f(x)$зЪДжЛРзВєгАВ

жЛРзВєжШѓеЗєеТМеЗЄжЫ≤зЇњзЪДеИЖзХМзВєгАВ

еЃЪзРЖ6.16 иЛ•$f$еЬ®$x_0$дЇМйШґеПѓеѓЉпЉМеИЩ$(x_0, f(x_0))$дЄЇжЫ≤зЇњ$y = f(x)$зЪДжЛРзВєзЪДењЕи¶БжЭ°дїґжШѓ$f^{(2)} = 0$.

еЃЪзРЖ6.17 иЃЊ$f$еЬ®$x_0$еПѓеѓЉпЉМеЬ®жЯРйҐЖеЯЯ$U^o(x_0)$дЄКдЇМйШґеПѓеѓЉгАВиЛ•еЬ®$U^o_{+}(x_0)$еТМ$U^o_{-}(x_0)$дЄК$f^{(2)}$зЪДзђ¶еПЈзЫЄеПНпЉМеИЩ$(x_0, f(x_0))$дЄЇжЫ≤зЇњ$y = f(x)$зЪДжЛРзВєгАВ

ж≥®жДПпЉЪиЛ•$(x_0, f(x_0))$жШѓжЫ≤зЇњ$y = f(x)$зЪДдЄАдЄ™жЛРзВєпЉМ$y = f(x)$еЬ®$x_0$зЪДеѓЉжХ∞дЄНдЄАеЃЪе≠ШеЬ®пЉМе¶В$y = \sqrt[3]{x}$еЬ®$x = 0$гАВ

еЃЪзРЖ е¶ВдЄЛдЄЙдЄ™йЩИињ∞жИРзЂЛгАВ

  2. иЛ•$f, g$еЭЗдЄЇеЗЄеЗљжХ∞пЉМеИЩ$f + g$дЄЇеЗЄеЗљжХ∞пЉЫ
  3. иЛ•$f$дЄЇеМЇйЧі$I$дЄКеЗЄеЗљжХ∞пЉМ$g$дЄЇ$J \supset f(I)$дЄКеЗЄеҐЮеЗљжХ∞пЉМеИЩ$g \circ f$дЄЇ$I$дЄКеЗЄеЗљжХ∞гАВ

еЃЪзРЖ иЃЊ$f$дЄЇеМЇйЧі$I$дЄКдЄ•ж†ЉеЗЄеЗљжХ∞пЉМ$x_0 \in I$дЄЇ$f$зЪДжЮБе∞ПеАЉзВєпЉМеИЩ$x_0$дЄЇ$f$еЬ®$I$дЄКеФѓдЄАзЪДжЮБе∞ПеАЉзВєгАВ

еЃЪзРЖ иЛ•$f, g$еЭЗдЄЇеМЇйЧі$I$дЄКеЗЄеЗљжХ∞пЉМеИЩ$F(x) = \max {f(x), g(x)}$дєЯжШѓ$I$дЄКеЗЄеЗљжХ∞гАВ

еЗ†дЄ™йЗНи¶БдЄНз≠ЙеЉП

еЭЗеАЉдЄНз≠ЙеЉП иЃЊ$a_i > 0 \ (i = 1, 2, \cdots, n)$пЉМжЬЙ
