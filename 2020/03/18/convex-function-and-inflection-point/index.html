еҮёеҮҪж•°е’ҢжӢҗзӮ№

еҸ‘иЎЁдәҺ2020-03-18|жӣҙж–°дәҺ2023-09-10|mathsmathematical analysis
|еӯ—ж•°жҖ»и®Ў:1.5k|йҳ…иҜ»ж—¶й•ҝ:6еҲҶй’ҹ|йҳ…иҜ»йҮҸ:

еҮёеҮҪж•°пјҲConvex functionпјүе’ҢжӢҗзӮ№пјҲInflection pointпјүгҖӮ жӯӨйғЁеҲҶдёәеҸӮиҖғ еҚҺдёңеёҲиҢғеӨ§еӯҰгҖҠж•°еӯҰеҲҶжһҗдёҠеҶҢгҖӢгҖӮд»ҘдёӢеҶ…е®№дёәеӯҰд№ дёӯзҡ„йҮҚзӮ№жҖ»з»“гҖӮ

еҮёеҮҪж•°дёҖиҲ¬з”ЁжқҘиҜҒжҳҺдёҚзӯүејҸпјӣжіЁж„Ҹе·ҰеҸіеҜјж•°е’ҢеҮёеҮҪж•°е®ҡд№үе…¬ејҸзҡ„иҒ”зі»пјӣеҮёеҮҪж•°зҡ„жҖ§иҙЁжҜ”иҫғеҘҪпјҢеңЁжңәеҷЁеӯҰд№ дёӯжҳҜзҗҶжғізҡ„д»Јд»·еҮҪж•°пјҢеӣ дёәеҘҪжұӮжңҖе°ҸеҖјзӮ№пјӣжіЁж„ҸжӢҗзӮ№дёӯзҡ„йһҚзӮ№пјҲжҳҜдёҖйҳ¶еҜјж•°зӯүдәҺ0зҡ„зӮ№пјҢеҚіжӢҗзӮ№дёӯзҡ„й©»зӮ№жҲ–зЁіе®ҡзӮ№пјүгҖӮ

еҮёеҮҪж•°

е®ҡд№ү1 и®ҫ$f$дёәе®ҡд№үеңЁеҢәй—ҙ$I$дёҠзҡ„еҮҪж•°пјҢиӢҘеҜ№$I$дёҠзҡ„д»»ж„ҸдёӨзӮ№$x_1, x_2$е’Ңд»»ж„Ҹе®һж•°$\lambda\in(0,1)$жҖ»жңү
$$
f(\lambda x_1 + (1-\lambda) x_2)\leq \lambda f(x_1) + (1 - \lambda)f(x_2),
$$
еҲҷз§°$f$дёә$I$дёҠзҡ„еҮёеҮҪж•°гҖӮеҸҚд№ӢпјҢеҰӮжһңжҖ»жңү
$$
f(\lambda x_1 + (1 - \lambda) x_2)\geq \lambda f(x_1) + (1 - \lambda) f(x_2),
$$
еҲҷз§°$f$дёә$I$дёҠзҡ„еҮ№еҮҪж•°гҖӮ

еҸҰеӨ–пјҢ

  1. еҰӮжһңдёҠйқўзҡ„дёҚзӯүејҸдёәдёҘж јдёҚзӯүејҸпјҢйӮЈд№Ҳз§°дёҘж јеҮёеҮҪж•°е’ҢдёҘж јеҮ№еҮҪж•°гҖӮ
  2. еҰӮжһң$-f$дёәеҢәй—ҙ$I$дёҠзҡ„еҮ№еҮҪж•°пјҢйӮЈд№Ҳ$f$дёәеҢәй—ҙ$I$дёҠзҡ„еҮёеҮҪж•°гҖӮ

еј•зҗҶ $f$дёә$I$дёҠзҡ„еҮёеҮҪж•°зҡ„е……иҰҒжқЎд»¶жҳҜеҜ№дәҺ$I$дёҠзҡ„д»»ж„ҸдёүзӮ№$x_1 < x_2 < x_3$пјҢжҖ»жңү
$$
\frac{f(x_2) - f(x_1)}{x_2 - x_1} \leq \frac{f(x_3) - f(x_2)}{x_3 - x_2}.
$$
еҗҢж ·ең°пјҢ$f$дёә$I$дёҠзҡ„еҮёеҮҪж•°зҡ„е……иҰҒжқЎд»¶жҳҜеҜ№дәҺ$I$дёҠзҡ„д»»ж„ҸдёүзӮ№$x_1 < x_2 < x_3$пјҢжңү
$$
\frac{f(x_2) - f(x_1)}{x_2 - x_1} \leq \frac{f(x_3) - f(x_1)}{x_3 - x_1} \leq \frac{f(x_3) - f(x_2)}{x_3 - x_2}.
$$
жіЁж„ҸпјҢиҝҷйҮҢжҠҠ$x_2, x_3$зңӢзқҖдёҖдёӘеҸҳйҮҸж—¶пјҢеҮҪж•°$\frac{f(x_3) - f(x_1)}{x_3 - x_1}$жҳҜдёҖдёӘеўһеҮҪж•°пјҢеҸҜд»ҘеҲ©з”ЁеҚ•и°ғжңүз•Ңе®ҡзҗҶпјҢеҸ–жһҒйҷҗ

е®ҡзҗҶ6.14 и®ҫ$f$дёәеҢәй—ҙ$I$дёҠзҡ„еҸҜеҜјеҮҪж•°пјҢеҲҷдёӢиҝ°и®әж–ӯдә’зӣёзӯүд»·пјҡ

  1. $f$дёә$I$дёҠзҡ„еўһеҮҪж•°пјӣ

  2. $f^{\prime}$дёә$I$дёҠзҡ„еўһеҮҪж•°пјӣ

  3. еҜ№$I$дёҠзҡ„д»»ж„ҸдёӨзӮ№$x_1, x_2$жңү
    $$
    f(x_2) \geq f(x_1) + f^{\prime}(x_1)(x_2 - x_1).
    $$
    и®әж–ӯ3зҡ„еҮ дҪ•ж„Ҹд№үе°ұжҳҜиҜҙжӣІзәҝ$y=f(x)$жҖ»жҳҜеңЁе®ғзҡ„д»»дёҖеҲҮзәҝдёҠж–№гҖӮ

е®ҡзҗҶ 6.15 и®ҫ$f$дёәеҢәй—ҙ$I$дёҠзҡ„дәҢйҳ¶еҸҜеҜјеҮҪж•°пјҢеҲҷеңЁ$I$дёҠ$f$дёәеҮёеҮҪж•°пјҲеҮ№еҮҪж•°пјүзҡ„е……иҰҒжқЎд»¶жҳҜ
$$
f^{(2)}(x) \geq 0 \ (f^{(2)}(x) \leq 0), \ x \in I.
$$
е®ҡзҗҶ иӢҘеҮҪж•°$f$дёәе®ҡд№үеңЁејҖеҢәй—ҙ$(a, b)$дёҠзҡ„еҸҜеҜјзҡ„еҮёеҮҪж•°пјҲеҮ№еҮҪж•°пјүпјҢеҲҷ$x_0 \in (a, b)$дёә$f$зҡ„жһҒе°ҸеҖјзӮ№пјҲжһҒеӨ§еҖјзӮ№пјүзҡ„е……иҰҒжқЎд»¶жҳҜ$x_0$дёә$f$зҡ„зЁіе®ҡзӮ№пјҢеҚі$f^{\prime}(x_0) = 0$.

е®ҡзҗҶ и®ҫеҮҪж•°$f(x)$дёәејҖеҢәй—ҙ$(a, b)$дёҠзҡ„еҮёеҮҪж•°пјҢдёҚжҒ’дёәеёёж•°пјҢеҲҷ$f(x)$дёҚеҸ–жңҖеӨ§еҖјгҖӮ

вҖӢ еҮёеҮҪж•°зҡ„жңҖеӨ§еҖјеҸӘиғҪеңЁз«ҜзӮ№еӨ„еҸ–еҫ—гҖӮеҚіпјҢиӢҘеҮҪж•°$f(x)$жҳҜй—ӯеҢәй—ҙ$[a, b]$дёҠзҡ„еҮёзҡ„иҝһз»ӯеҮҪж•°пјҢйӮЈд№Ҳ
$$
f(x) \leq \max{f(a), f(b)}.
$$
и©№жЈ®пјҲJensenпјүдёҚзӯүејҸ иӢҘ$f$дёә$[a, b]$дёҠзҡ„еҮёеҮҪж•°пјҢеҲҷеҜ№д»»ж„Ҹ$x_i \in [a, b], \lambda_i > 0 \ (i = 1, 2, \cdots, n), \ \sum^n_{i = 1} \lambda_i = 1$пјҢжңү
$$
f(\sum^n_{i=1} \lambda_i x_i) \leq \sum^n_{i=1} \lambda_i f(x_i).
$$
еј•зҗҶ и®ҫ$f$дёәејҖеҢәй—ҙ$I$еҶ…зҡ„еҮёеҮҪж•°пјҲеҮ№еҮҪж•°пјүпјҢеҲҷ$f$еңЁ$I$еҶ…д»»дёҖзӮ№$x_0$йғҪеӯҳеңЁе·ҰгҖҒеҸіеҜјж•°гҖӮд»ҺиҖҢ$f$еңЁ$x_0$иҝһз»ӯгҖӮ

жӢҗзӮ№

е®ҡд№ү2 и®ҫжӣІзәҝ$y = f(x)$еңЁзӮ№$(x_0, f(x_0))$еӨ„жңүз©ҝиҝҮжӣІзәҝзҡ„еҲҮзәҝпјҢдё”еңЁеҲҮзӮ№иҝ‘ж—ҒпјҢжӣІзәҝеңЁеҲҮзәҝзҡ„дёӨдҫ§еҲҶеҲ«жҳҜдёҘж јеҮёе’ҢдёҘж јеҮ№зҡ„пјҢиҝҷж—¶з§°зӮ№$(x_0, f(x_0))$дёәжӣІзәҝ$y = f(x)$зҡ„жӢҗзӮ№гҖӮ

жӢҗзӮ№жҳҜеҮ№е’ҢеҮёжӣІзәҝзҡ„еҲҶз•ҢзӮ№гҖӮ

е®ҡзҗҶ6.16 иӢҘ$f$еңЁ$x_0$дәҢйҳ¶еҸҜеҜјпјҢеҲҷ$(x_0, f(x_0))$дёәжӣІзәҝ$y = f(x)$зҡ„жӢҗзӮ№зҡ„еҝ…иҰҒжқЎд»¶жҳҜ$f^{(2)} = 0$.

е®ҡзҗҶ6.17 и®ҫ$f$еңЁ$x_0$еҸҜеҜјпјҢеңЁжҹҗйўҶеҹҹ$U^o(x_0)$дёҠдәҢйҳ¶еҸҜеҜјгҖӮиӢҘеңЁ$U^o_{+}(x_0)$е’Ң$U^o_{-}(x_0)$дёҠ$f^{(2)}$зҡ„з¬ҰеҸ·зӣёеҸҚпјҢеҲҷ$(x_0, f(x_0))$дёәжӣІзәҝ$y = f(x)$зҡ„жӢҗзӮ№гҖӮ

жіЁж„ҸпјҡиӢҘ$(x_0, f(x_0))$жҳҜжӣІзәҝ$y = f(x)$зҡ„дёҖдёӘжӢҗзӮ№пјҢ$y = f(x)$еңЁ$x_0$зҡ„еҜјж•°дёҚдёҖе®ҡеӯҳеңЁпјҢеҰӮ$y = \sqrt[3]{x}$еңЁ$x = 0$гҖӮ

е®ҡзҗҶ еҰӮдёӢдёүдёӘйҷҲиҝ°жҲҗз«ӢгҖӮ

  1. иӢҘ$f$дёәеҮёеҮҪж•°пјҢ$\lambda$дёәйқһиҙҹе®һж•°пјҢеҲҷ$\lambda f$дёәеҮёеҮҪж•°пјӣ
  2. иӢҘ$f, g$еқҮдёәеҮёеҮҪж•°пјҢеҲҷ$f + g$дёәеҮёеҮҪж•°пјӣ
  3. иӢҘ$f$дёәеҢәй—ҙ$I$дёҠеҮёеҮҪж•°пјҢ$g$дёә$J \supset f(I)$дёҠеҮёеўһеҮҪж•°пјҢеҲҷ$g \circ f$дёә$I$дёҠеҮёеҮҪж•°гҖӮ

е®ҡзҗҶ и®ҫ$f$дёәеҢәй—ҙ$I$дёҠдёҘж јеҮёеҮҪж•°пјҢ$x_0 \in I$дёә$f$зҡ„жһҒе°ҸеҖјзӮ№пјҢеҲҷ$x_0$дёә$f$еңЁ$I$дёҠе”ҜдёҖзҡ„жһҒе°ҸеҖјзӮ№гҖӮ

е®ҡзҗҶ иӢҘ$f, g$еқҮдёәеҢәй—ҙ$I$дёҠеҮёеҮҪж•°пјҢеҲҷ$F(x) = \max {f(x), g(x)}$д№ҹжҳҜ$I$дёҠеҮёеҮҪж•°гҖӮ

еҮ дёӘйҮҚиҰҒдёҚзӯүејҸ

еқҮеҖјдёҚзӯүејҸ и®ҫ$a_i > 0 \ (i = 1, 2, \cdots, n)$пјҢжңү
$$
\frac{n}{\frac{1}{a_1} + \frac{1}{a_2} + \cdots + \frac{1}{a_n}} \leq \sqrt[n]{a_1 a_2 \cdots a_n} \leq \frac{a_1 + a_2 + \cdots + a_n}{n}
$$
иҜҒжҳҺпјҢжһ„йҖ еҮҪж•°$f(x) = \ln(x)$еҸҜиҜҒжҳҺз¬¬дәҢдёӘдёҚзӯүеҸ·пјҢз„¶еҗҺз”ЁеҸҳеҢ–$x = \frac{1}{t}$е’Ңз¬¬дәҢдёӘдёҚзӯүеҸ·иҜҒжҳҺз¬¬дёҖдёӘдёҚзӯүеҸ·гҖӮ

Young дёҚзӯүејҸ и®ҫ$a, b, p, q$еқҮжҳҜжӯЈе®һж•°пјҢдё”$\frac{1}{p} + \frac{1}{q} = 1$пјҢеҲҷ
$$
ab \leq \frac{a^p}{p} + \frac{b^q}{q}
$$
зӯүеҸ·жҲҗз«ӢеҪ“дё”д»…еҪ“ $a^p = b^q$.

иҜҒжҳҺпјҢеҲ©з”ЁеҮҪж•°$y = e^x$зҡ„еҮёжҖ§
$$
ab = e^{\ln(a)} e^{\ln(b)} = e^{\frac{1}{p} \ln(a^p) + \frac{1}{q} \ln(b^q)} \leq \frac{1}{p} e^{\ln(a^p)} + \frac{1}{q} e^{\ln(b^q)} = \frac{a^p}{p} + \frac{b^q}{q}.
$$
еҸҜд»Ҙз”ЁиҜҘдёҚзӯүејҸиҜҒжҳҺеҰӮдёӢзҡ„HolderдёҚзӯүејҸгҖӮ

Holder дёҚзӯүејҸ и®ҫ$a_i, b_i > 0 \ (i = 1, 2, \cdots, n)$пјҢжңү
$$
\sum^n_{i = 1} a_i b_i \leq (\sum^n_{i = 1} a^p_i)^{\frac{1}{p}}(\sum^n_{i = 1} b^q_i)^{\frac{1}{q}}
$$
е…¶дёӯ$p > 1, q > 1, \frac{1}{p} + \frac{1}{q} = 1$.

иҜҒжҳҺпјҢеҸ–$a = \frac{a_k}{(\sum^n_{i = 1}a^p_i)^{\frac{1}{p}}}$, $b = \frac{b_k}{(\sum^n_{i = 1}b^q_i)^{\frac{1}{q}}}$ пјҢе№¶еёҰе…ҘYoungдёҚзӯүејҸгҖӮ
