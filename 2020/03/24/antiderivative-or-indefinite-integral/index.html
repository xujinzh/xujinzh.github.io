дёҚе®ҡз§ҜеҲҶ

еҸ‘иЎЁдәҺ2020-03-24|жӣҙж–°дәҺ2026-04-26|mathsmathematical analysis
|еӯ—ж•°жҖ»и®Ў:1k|йҳ…иҜ»ж—¶й•ҝ:4еҲҶй’ҹ|йҳ…иҜ»йҮҸ:

з§ҜеҲҶжі•еұһдәҺеҫ®еҲҶжі•зҡ„йҖҶиҝҗз®—пјҢе°ұеғҸеҮҸжі•жҳҜеҠ жі•зҡ„йҖҶиҝҗз®—пјҢйҷӨжі•жҳҜд№ҳжі•зҡ„йҖҶиҝҗз®—дёҖж ·гҖӮ

еҺҹеҮҪж•°дёҺдёҚе®ҡз§ҜеҲҶ

е®ҡд№ү 1 &emsp;В и®ҫеҮҪж•°$f$дёҺ$F$еңЁеҢәй—ҙ$I$дёҠйғҪжңүе®ҡд№үгҖӮиӢҘ
$$
F^{\prime}(x) = f(x), \ x \in I,
$$
еҲҷз§°$F$дёә$f$еңЁеҢәй—ҙ$I$дёҠзҡ„дёҖдёӘеҺҹеҮҪж•°гҖӮ

иҝҷдёҖиҠӮи§ЈеҶіж»Ўи¶ід»Җд№Ҳж ·жқЎд»¶зҡ„еҮҪж•°еӯҳеңЁеҺҹеҮҪж•°гҖӮ

е®ҡзҗҶ 8.1 &emsp; иӢҘеҮҪж•°$f$еңЁеҢәй—ҙ$I$дёҠиҝһз»ӯпјҢеҲҷ$f$еңЁ$I$дёҠеӯҳеңЁеҺҹеҮҪж•°$F$пјҢеҚі$F^{\prime} = f(x), \ x \in I$.

еҲқзӯүеҮҪж•°еңЁе®ҡд№үеҹҹеҶ…жҳҜиҝһз»ӯеҮҪж•°пјҢеӣ жӯӨпјҢеҲқзӯүеҮҪж•°йғҪеӯҳеңЁеҺҹеҮҪж•°гҖӮжіЁж„ҸпјҢеҲқзӯүеҮҪж•°зҡ„еҺҹеҮҪж•°дёҚдёҖе®ҡиҝҳжҳҜеҲқзӯүеҮҪж•°гҖӮ

е®ҡзҗҶ 8.2 &emsp;В и®ҫ$F$жҳҜ$f$еңЁеҢәй—ҙ$I$дёҠзҡ„дёҖдёӘеҺҹеҮҪж•°пјҢеҲҷ

  1. $F + C$ д№ҹжҳҜ$f$еңЁ$I$дёҠзҡ„еҺҹеҮҪж•°пјҢе…¶дёӯ$C$дёәд»»ж„ҸеёёйҮҸеҮҪж•°пјӣ
  2. $f$еңЁ$I$дёҠзҡ„д»»ж„ҸдёӨдёӘеҺҹеҮҪж•°д№Ӣй—ҙпјҢеҸӘеҸҜиғҪзӣёе·®дёҖдёӘеёёж•°гҖӮ

е®ҡд№ү 2 &emsp; еҮҪж•°$f$еңЁеҢәй—ҙ$I$дёҠзҡ„е…ЁдҪ“еҺҹеҮҪж•°з§°дёә$f$еңЁ$I$дёҠзҡ„дёҚе®ҡз§ҜеҲҶпјҢи®°дҪң
$$
\int f(x) \mathrm{d}x,
$$
е…¶дёӯз§°$\int$дёәз§ҜеҲҶеҸ·пјҢ$f(x)$дёәиў«з§ҜеҮҪж•°пјҢ$f(x)\mathrm{d}x$дёәиў«з§ҜиЎЁиҫҫејҸпјҢ$x$дёәз§ҜеҲҶеҸҳйҮҸгҖӮе°Ҫз®ЎдёҠиҝ°и®°еҸ·дёӯеҗ„дёӘйғЁеҲҶйғҪжңүе…¶зү№е®ҡзҡ„еҗҚз§°пјҢдҪҶеңЁдҪҝз”Ёж—¶еҝ…йЎ»жҠҠе®ғд»¬зңӢдҪңдёҖдёӘж•ҙдҪ“гҖӮ

дёҚе®ҡз§ҜеҲҶдёҺеҺҹеҮҪж•°жҳҜжҖ»дҪ“дёҺдёӘдҪ“зҡ„е…ізі»пјҢеҚіиӢҘ$F$жҳҜ$f$зҡ„дёҖдёӘеҺҹеҮҪж•°пјҢеҲҷ$f$зҡ„дёҚе®ҡз§ҜеҲҶжҳҜдёҖдёӘеҮҪж•°ж—Ҹ$\{F + C\}$пјҢе…¶дёӯ$C$жҳҜд»»ж„Ҹеёёж•°пјҢдёәдәҶж–№дҫҝеёёеҶҷдҪң
$$
\int f(x) \mathrm{d}x = F(x) + C.
$$
з§°$C$дёәз§ҜеҲҶеёёж•°пјҢе®ғеҸҜеҸ–д»»дёҖе®һж•°еҖјгҖӮ

дёҚе®ҡз§ҜеҲҶзҡ„еҮ дҪ•ж„Ҹд№ү &emsp;В иӢҘ$F$жҳҜ$f$зҡ„дёҖдёӘеҺҹеҮҪж•°пјҢеҲҷз§°$y = F(x)$зҡ„еӣҫеғҸдёә$f$зҡ„дёҖжқЎз§ҜеҲҶжӣІзәҝгҖӮдәҺжҳҜпјҢ$f$зҡ„дёҚе®ҡз§ҜеҲҶеңЁеҮ дҪ•дёҠиЎЁзӨә$f$зҡ„жҹҗдёҖз§ҜеҲҶжӣІзәҝжІҝзәөиҪҙж–№еҗ‘д»»ж„Ҹе№із§»жүҖеҫ—дёҖеҲҮз§ҜеҲҶжӣІзәҝз»„жҲҗзҡ„жӣІзәҝж—ҸгҖӮжҳҫз„¶пјҢиӢҘеңЁжҜҸдёҖжқЎз§ҜеҲҶжӣІзәҝдёҠжЁӘеқҗж ҮзӣёеҗҢзҡ„зӮ№еӨ„дҪңеҲҮзәҝпјҢеҲҷиҝҷдәӣеҲҮзәҝдә’зӣёе№іиЎҢгҖӮ

еҪ“з»ҷе®ҡеҺҹеҮҪж•°зҡ„жҹҗдёҖдёӘеҲқе§ӢжқЎд»¶еҗҺпјҢеҸҜд»ҘжұӮеҮәж»Ўи¶іиҜҘеҲқе§ӢжқЎд»¶зҡ„жҹҗдёҖеҺҹеҮҪж•°еӯҗйӣҶеҗҲгҖӮ

еҹәжң¬з§ҜеҲҶиЎЁ

жұӮеҜјж•°зӣёеҜ№дәҺжұӮз§ҜеҲҶдјҡе®№жҳ“пјҢеҜ№дәҺжҹҗдәӣеҹәжң¬зҡ„з§ҜеҲҶе…¬ејҸеҲ—еҮәеҰӮдёӢпјҡ

  1. $\int 0 \mathrm{d}x = C$;
  2. $\int 1 \mathrm{d}x = \int \mathrm{d}x = x + C$;
  3. $\int x^{\alpha} \mathrm{d}x = \frac{x^{\alpha + 1}}{\alpha + 1} + C \ (\alpha \neq -1, \ x > 0)$;
  4. $\int \frac{1}{x} \mathrm{d}x = \ln |x| + C \ (x \neq 0)$;
  5. $\int e^x \mathrm{d}x = e^x + C$;
  6. $\int a^x \mathrm{d}x = \frac{a^x}{\ln a} + C \ (a > 0, \ a \neq 1)$;
  7. $\int \cos ax \mathrm{d}x = \frac{1}{a} \sin ax + C \ (a \neq 0)$;
  8. $\int \sin ax \mathrm{d}x = - \frac{1}{a} \cos ax + C \ (a \neq 0)$;
  9. $\int \sec^2 x \mathrm{d}x = \tan x + C$;
  10. $\int \csc^2 x \mathrm{d}x = -\cot x + C$;
  11. $\int \sec x \cdot \tan x \mathrm{d}x = \sec x + C$;
  12. $\int \csc x \cdot \cot x \mathrm{d}x = -\csc x + C$;
  13. $\int \frac{1}{\sqrt[2]{1 - x^2}} \mathrm{d}x = \arcsin x + C = -\arccos x + C$;
  14. $\int \frac{1}{1 + x^2} \mathrm{d}x = \arctan x + C = -\text{arccot}\ x + C_1$.

**е®ҡзҗҶ 8.3 **&ensp;В иӢҘеҮҪж•°$f$дёҺ$g$еңЁеҢәй—ҙ$I$дёҠйғҪеӯҳеңЁеҺҹеҮҪж•°пјҢ$k_1, k_2$дёәдёӨдёӘд»»ж„Ҹеёёж•°пјҢеҲҷ$k_1 f + k_2 g$еңЁ$I$дёҠд№ҹеӯҳеңЁеҺҹеҮҪж•°пјҢдё”еҪ“$k_1$дёҺ$k_2$дёҚеҗҢж—¶дёәйӣ¶ж—¶жңү
$$
\int [k_1 f(x) + k_2 g(x)] \mathrm{d}x = k_1 \int f(x) \mathrm{d}x + k_2 \int g(x) \mathrm{d}x.
$$
дёҖдәӣдәӢе®һпјҡ

жҜҸдёҖдёӘеҗ«жңүз¬¬дёҖзұ»й—ҙж–ӯзӮ№зҡ„еҮҪж•°йғҪжІЎжңүеҺҹеҮҪж•°гҖӮ

еҜјеҮҪж•°дёҚеӯҳеңЁз¬¬дёҖзұ»й—ҙж–ӯзӮ№гҖӮ

жҹҗдәӣеҜјеҮҪж•°еҸҜиғҪеӯҳеңЁз¬¬дәҢзұ»й—ҙж–ӯзӮ№гҖӮеҰӮдёӢеҮҪж•°зҡ„еҜјеҮҪж•°$f(x)$еңЁ$x = 0$еӨ„жҳҜз¬¬дәҢзұ»й—ҙж–ӯзӮ№
$$
F(x) =
\begin{cases}
x^{3/2} \sin \frac{1}{x}, \quad x \in (0, 1]; \\
0, \quad \quad \quad x = 0.
\end{cases}
$$
