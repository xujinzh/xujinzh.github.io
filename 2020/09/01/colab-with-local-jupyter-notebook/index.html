Colab 使用本地 Jupyter Notebook 内核运行时

发表于 2020-09-01 更新于 2022-01-20 分类于 technology ， python 阅读次数： 本文字数： 1.5k 阅读时长 ≈ 1 分钟

众所周知，Colab 是一个美观、工具齐全、编写 Python 代码方便的网页版 notebook，结合本地运行时能够提高代码开发效率。注意本篇使用的 Jupyterlab 版本小于3.

设置前注意事项：

  • 调用任意命令（例如“rm -rf /”）
  • 访问本地文件系统
  • 在计算机上运行恶意内容
